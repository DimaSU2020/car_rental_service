# Архитектурные контракты

В этом каталоге содержатся **архитектурные контракты** для основных сценариев использования.

Каждый контракт определяет:
- поток запросов между уровнями
- границы ответственности
- семантику ошибок
- гарантии идемпотентности

Эти документы являются нормативными.
Изменения в коде должны сохранять эти контракты или обновлять их явным образом.
